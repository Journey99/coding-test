## BOJ(Baekjoon Online Judge)

문제는 백준에서 제공하는 [단계별](https://www.acmicpc.net/step)로 진행
  
```
STEP1 : 입출력과 사칙연산
STEP2 : 조건문
STEP3 : 반복문
STEP4 : 배열
STEP5 : 문자열
STEP6 : 심화1
STEP7 : 2차원 배열
STEP8 : 일반 수학1
STEP9 : 약수, 배수와 소수
STEP10 : 기하: 직사각형과 삼각형
STEP11 : 시간 복잡도
STEP12 : 브루트 포스
STEP13 : 정렬
STEP14 : 집합과 맵
STEP15 : 약수, 배수와 소수 2
```
